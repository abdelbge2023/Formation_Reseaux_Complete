README - Plateforme pédagogique Réseaux
=================================

Contenu du package:
- index.html : site statique complet (cours, labs, exercices, simulateur, quiz, dashboard local).

Instructions:
1. Dézippez le fichier et ouvrez index.html dans un navigateur moderne (Chrome/Firefox).
2. Pour utiliser le tableau de bord et sauvegarder la progression, utilisez le même navigateur (les données sont stockées en localStorage).
3. Pour un déploiement en production, hébergez les fichiers sur un serveur HTTPS et ajoutez un backend pour la gestion utilisateurs (optionnel).

Notes techniques:
- Le simulateur et la logique de correction sont entièrement côté client (JavaScript).
- Pour multi-utilisateurs et stockage centralisé, prévoir API + base de données.
